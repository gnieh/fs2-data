  9.  class SelectorParser[F[_]] extends AnyRef

    Parses a filter string.

    Parses a filter string. Syntax is as follows:

    Selector ::= `.`
           | Sel Sel*

Sel ::= `.` Name `!`? `?`?
      | `.` `[` String (`,` String)* `]` `!`? `?`?
      | `.` `[` Integer (`,` Integer)* `]` `?`?
      | `.` `[` Integer `:` Integer `]` `?`?
      | `.` `[` `]` `?`?

Name ::= [a-zA-Z_][a-zA-Z0-9_]*

String ::= <a json string>

Integer ::= 0
          | [1-9][0-9]*
